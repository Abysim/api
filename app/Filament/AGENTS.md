# Filament

## Purpose
Filament v3 admin panel resources for managing News and FlickrPhoto records through a web UI.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `Resources/` | Filament resource definitions with table/form schemas (see `Resources/AGENTS.md`) |

## For AI Agents

### Working In This Directory
- Filament admin is accessible at `/admin` on the bigcats server
- Resources define table columns, filters, forms, and page layouts
- The News resource URL is used in `News::getCaption()` for edit links during review
